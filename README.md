# 📸 Poto Ku - Portfolio Fotografer Professional

> Website portofolio fotografi modern, responsif, dan siap deploy ke GitHub Pages.  
> **Dibuat oleh Muhammad Fahreza Ramadhan**

## 🖼️ Tentang Project

**Poto Ku** adalah website one-page untuk fotografer yang menampilkan:
- Hero section dengan ajakan aksi
- Galeri foto (grid interaktif dengan efek hover)
- Tentang fotografer
- Layanan yang ditawarkan
- Testimoni klien
- Form kontak & tautan media sosial (termasuk LinkedIn aktif)
- Navigasi smooth scroll & mobile friendly

✨ **Setiap bagian dipisah secara struktural** (Hero, Galeri, Tentang, Layanan, Testimoni, Kontak) untuk memudahkan pengembangan lebih lanjut.

## 🚀 Cara Deploy ke GitHub Pages

Ikuti langkah-langkah berikut untuk menampilkan website ini secara online GRATIS melalui GitHub Pages:

### 1. Buat Repository Baru di GitHub
- Buka [github.com](https://github.com) dan login
- Klik tombol **New** repository
- Beri nama misalnya: `poto-ku` atau `fotografer-portfolio`

### 2. Upload File ke Repository
Ada dua cara:

**Cara A (langsung via browser):**
- Download file `index.html` (kode website yang sudah diberikan)
- Di repository GitHub, klik **Add file** → **Upload files**
- Upload file `index.html`
- Upload juga file `README.md` ini (copy teks ini ke file README.md)

**Cara B (via terminal/git):**
```bash
git clone https://github.com/username/poto-ku.git
cd poto-ku
# masukkan file index.html dan README.md ke folder ini
git add .
git commit -m "Initial commit - Poto Ku portfolio fotografer"
git push origin main
